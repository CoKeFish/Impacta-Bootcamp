# Frontend - Stellar Scaffold

Este directorio contendrá el frontend creado con Stellar Scaffold.

## Inicializar con Stellar Scaffold

Desde este directorio, ejecuta:

```bash
npx create-soroban-dapp@latest .
```

O si prefieres hacerlo desde Docker:

```bash
docker run -it --rm -v ${PWD}:/app -w /app node:20-alpine sh -c "npx create-soroban-dapp@latest ."
```

Después de crear el proyecto, el contenedor de Docker podrá iniciarlo automáticamente.
